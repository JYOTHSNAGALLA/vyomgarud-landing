# VyomGarud Landing Page

A modern, responsive landing page for VyomGarud, a UAV/drone systems company.

## Setup
1. Clone repo: `git clone <repo-url>`
2. Install: `npm install`
3. Run: `npm run dev`
4. Open `http://localhost:3000`

## Design Notes
- Dark, military-grade theme with charcoal (#0b0c10), white, and orange (#ff7b00) colors.
- Fonts: Montserrat for headings, Inter for body.
- Built with Next.js, Tailwind CSS, Framer Motion for animations.
- Inspired by onomondo.com and raphe.com.
